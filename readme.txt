== Description ==

Plugins para agregar las regiones a woocommerce y ocultar el codigo postal

*   Agrega lista de Regiones
*   Oculta el campo Codigo postal

*   IX La Araucanía
*   XIV Los Ríos
*   X Los Lagos
*   XI Aysén
*   XII Magallanes y Antártica
*   Autoridades
